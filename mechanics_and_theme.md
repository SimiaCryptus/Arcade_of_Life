These games are 2d arcade-style simple games, generally inspired by simple arcade classics.

Game mechanics are implemented via the game of life (Conway's Game of Life) and cellular automata, which allows for
emergent gameplay and complex interactions from simple rules.

For example, the games will make heavy use of gliders and use life to simulate collisions, movement, and interactions
between game entities.

Player controls will need to have exceptions to allow for player input to override the game of life rules, but the core
mechanics will be based on cellular automata.
